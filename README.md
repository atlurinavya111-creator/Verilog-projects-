# Verilog HDL Projects

A collection of Verilog HDL projects developed for Digital Design and VLSI learning.

## Tools Used
- Verilog HDL
- Quartus Prime
- ModelSim

## Projects

### 1. Half Adder
- Performs binary addition of two bits
- Includes testbench

### 2. Full Adder
- Performs binary addition with carry input
- Includes testbench

### 3. 4-bit Ripple Carry Adder
- Built using Full Adders
- Includes testbench

### 4. 2x1 Multiplexer
- Data selection circuit
- Includes testbench

### 5. 2-to-4 Decoder
- Converts 2-bit input into 4 outputs
- Includes testbench

### 6. 4-to-2 Encoder
- Encodes 4 inputs into 2-bit output
- Includes testbench

### 7. 4-bit ALU
Operations:
- Addition
- Subtraction
- AND
- OR

Includes testbench.

### 8. FIFO
Features:
- Read operation
- Write operation
- Full flag
- Empty flag

Includes testbench.

### 9. UART Transmitter
- Serial data transmission
- Includes testbench
