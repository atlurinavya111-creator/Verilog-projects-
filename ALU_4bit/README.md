# 4-bit ALU

## Description

This project implements a 4-bit Arithmetic Logic Unit.

## Operations

| Select | Operation |
|----------|----------|
| 00 | Addition |
| 01 | Subtraction |
| 10 | AND |
| 11 | OR |

## Files

- alu_4bit.v
- alu_4bit_tb.v

## Tools Used

- Verilog HDL
- Quartus Prime
- ModelSim
